# 🎯 AWS Serverless Website Monitoring System - Project Summary

## Short Explanation

**What is it?**
A serverless website monitoring system that automatically checks if your websites are online and fast. When a site goes down or gets slow, it sends you an instant email alert.

**How it works?**
1. Every 5 minutes → AWS Lambda runs automatically
2. Checks if your websites respond → Measures response time
3. Site down or too slow? → Sends email alert instantly
4. All activity logged → For tracking and compliance

**Why use it?**
- ✅ **No Servers**: Nothing to manage or maintain
- ✅ **Always On**: Works 24/7 automatically
- ✅ **Cheap**: Free tier ($0.20/month in production)
- ✅ **Fast Alerts**: Know about issues in seconds
- ✅ **Enterprise Quality**: Uses AWS's reliable services

**Tech Stack:**
AWS Lambda + Python + SNS + EventBridge + CloudWatch + GitHub Actions
